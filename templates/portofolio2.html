Project Pertama - Deployment Model ML Amateur

Penulis: Rahmat Aldino

Tangal 16 April 2022

Pada project yang dimaksud ini adalah tugas pertama orbit materi technical. Materi yang diangkat adalah deployment model Artificial Intelegence menggunakan Restfull API flask modul pada tampilan antarmuka berbasis web. Model yang di training pada dataset tinggi dan berat badan suatu gender. Target dari model adalah memprediksi berat badan berdasarkan tinggi dan jenis gender. Algoritma machine learning yan digunakan adalah linear regression.
